    {{ site.data.site_config.site_abbrev }} Database

    The {{ site.data.site_config.site_title }} includes data from a variety of governmental and other public sources, unified in a freely distributed database. The tables from this database are also provided individually as csv files at the links below.

    Download the {{ site.data.site_config.site_abbrev }} SQLite unified database. SQLite databases are a simple and efficient format for distributing queryable datasets. You can read more about sqlitehere and read about how to interact with SQLite databases in the R and python languages here. Each analysis on this site draws directly from this database, so the open source scripts underlying those analyses can also serve as tutorials for using the database.

    Test a live demo of the {{ site.data.site_config.site_abbrev }} database in your browser. You can execute sample queries and generate results from the database on this page, without the need for any special software.
